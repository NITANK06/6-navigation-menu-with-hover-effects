
** Project Overview**

This project demonstrates a horizontal navigation menu styled with CSS.  
It includes:
 Hover states with smooth transitions
 Active state highlighting
 Clean, modern layout using flexbox



  Features
 **Horizontal menu** with evenly spaced items
 **Hover effect**: background color change + smooth transition
 **Active state**: highlights the current page link
 **Responsive design**: menu adapts to different screen sizes
